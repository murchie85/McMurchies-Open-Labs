# Python Tips  
    

### Capitalize  

It makes the first letter uppercase.  

```
txt = "python is awesome!"

txt.capitalize()
'Python is awesome!'
```


###   Upper

It makes all the letters uppercase.



```
txt = "Python is awesome!"

txt.upper()
'PYTHON IS AWESOME!'

```
   


###   Lower


It makes all the letters lowercase.


```
txt = "PYTHON IS AWESOME!"

txt.lower()
'python is awesome!'
```
   

###   Isupper

It checks if all the letters are uppercase.


```
txt = "PYTHON IS AWESOME!"

txt.isupper()
True
```
